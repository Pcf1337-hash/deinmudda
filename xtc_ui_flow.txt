Integration Points:
• Home Screen ← XTC button in quick entry bar
• Entry List ← Shows XTC entries like normal entries
• Statistics ← XTC entries count in substance statistics
• Export ← XTC data included in exports
• Search ← Can search XTC entries by name/notes

NOT Integrated:
• Substance Manager ← XTC substances don't appear here
• Regular Add Entry ← Uses separate specialized dialog
• Substance Selection ← XTC bypasses substance selection

Key Features Demonstrated:
✅ All 8 form options with appropriate icons
✅ All 4 content types with icons  
✅ mg-only dosage input with validation
✅ Square color preview → round color palette
✅ Bruchrillen switch with clear labels
✅ Optional weight field
✅ Clean, uncluttered layout
✅ Integration as quick button only
✅ Specialized data storage in notes
